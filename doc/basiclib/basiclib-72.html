Using the SystemEnv Fragment

A program using the systemEnv fragment will have the following structure:


INCLUDE '~beta/basiclib/systemenv'
--- program: descriptor ---
systemenv
(# process: @ |system(# ... #);
do ...
process[] -> fork;
...
#)
