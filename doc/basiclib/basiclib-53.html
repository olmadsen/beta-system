The regexp Library

Using the regexp Fragment

A program using the regexp fragment will have the following structure:

INCLUDE '~beta/basiclib/regexp
--- program: descriptor ---
(# ...
t: ^text
do ...
'Hello world' -> t[];
('\\<w.*\\>', '\\0 champion') -> t.regexp_replace;
t[] -> putline; (* prints 'Hello world champion' *)
...
#)
