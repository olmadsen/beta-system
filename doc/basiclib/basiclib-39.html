The formatio Library

Using the formatio Fragment

A program using the formatio fragment will have the following structure:


INCLUDE '~beta/basiclib/formatio
--- program: descriptor ---
(# ...
r: @real;
do ...
'real value: %e' -> putformat(# do r->e #);
...
#)
