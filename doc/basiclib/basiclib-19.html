The Stream Pattern

Substreams

There is an additional library called substreams.bet, which implements a substrems concept (and a subtext concept). A substream refers to a (consequtive) portion of another stream. Manipulations on the substream will thereby actually change that portion of this other stream. All usual stream operations applies to a substream. The interface af substream is not included in this manual.
