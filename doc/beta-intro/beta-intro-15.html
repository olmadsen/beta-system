Simple Assignments

Value assignment

Simple value assignments in BETA goes left to right:

2 -> i         (* assign the value 2 to i *)
i -> j         (* assign the value of i to j *)
i*j -> k       (* assign the value of i*j to k *)
(i,j) -> (x,y) (* assign the value of i to x and 
                *        the value of j to y 
                *)
