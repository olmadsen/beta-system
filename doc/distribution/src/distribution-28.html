CreateShell dependencies

For the Ensemble operation createShell to work, the owner of the executing process must have execution rights to the script

BETALIB/distribution/private/external/remoteStart

and the

BETALIB/distribution/private/external/$/startAsDeamon

executable on the target host. By default these executables are installed with execution rights for everyone.

remoteStart is a simple shell script determining the current machine type before invoking the machine specific startAsDeamonexecutable, which in turn executes the shell executable as a deamon. remoteStart is executed using rsh (remsh on HP UX platforms) on remote network hosts, or executed directly by the distributed BETA process, in case the new shell is started on the local host. If rsh is not found in the standard directory for the platform, the rshPath virtual described in section 5.5 can be used to tell shellEnv about the location of the rsh executable.
