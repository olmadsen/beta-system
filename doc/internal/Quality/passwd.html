Ændring af beta password

Sidst ændret: 7 August 2006.

Hvis man retter beta passwd på daimi skal man rette det på mjolner også, samt rette de to tilsvarende Web passwords. Passwordet på Daimi må ikke være identisk med password på Mjølner.

Gør f.eks. således: 


daimi> ssh www -l beta
daimi> passwd
daimi> cd ~beta/public_html/
daimi> htpasswd .htpasswd beta


sigurd: User sigurd har en variant af beta's passwd. 
Ret det tilsvarende (%).


mjolner> ssh www -l beta
mjolner> yppasswd (variant: #)
mjolner> cd ~beta/public_html/
mjolner> htpasswd2 .htpasswd beta

På Mjølners Mac (udgaard) rettes rod password tilsvarende (både i MacOS classic og MacOS X).

For de indviede vil følgende give mening: AriUlt, AriCol, ZenDui, ChaLot, CouPol, CouSin, CouVer, ParFai, ParDon, ParLeu, LbisXP, OrieXP, WindXp, ProgXp, ClrJvm, JvmClr, BarLon, LauSan, PalPre, PalBet, PreVMC, HamMer, HamBer.
