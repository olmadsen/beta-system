BETA Compiler Internal Documentation

Various internal aspects of the BETA compiler is documented from the directory compiler/DOC inside the compiler source directory. A mirror of this directory can be found at the URL http://www.cs.au.dk/~olm/DOC/.

A number of the things documented there may be of general interest. For this purpose, direct links are available below.

  • Low Level Primitives (new style replacing TOS)
     What is not included in this document is a table of how to transform the old style TOS calls to new style - i.e. what new primitives correspond to the old ones. This is not always obvious. To get an idea, check out the CVS project TST and compare the two files tstoldprim.bet and tstnewprim.bet. These are almost identical except for the change of primitive-style.
    Jørgen is working on a table for specifying this, this can be seen in TOSconvertTable.html.
