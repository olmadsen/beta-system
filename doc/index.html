Browser Requirements:

To get the most out of these manual pages, you should use a HTML browser with support for
  • HTML 3.2
  • Cascading Style Sheets, version 1 (CSS1)
  • JavaScript
These requirements are met by e.g. Netscape Navigator 4.0 (or later) and Microsoft Internet Explorer 4.0 (or later). Make sure you have Cascading Style Sheets and JavaScript enabled in your browser preferences.

If your browser does not meet the above requirements, the pages may look less pleasant, but should still be readable.
