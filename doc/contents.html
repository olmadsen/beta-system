Contents of MIADOC

This page contains links to all of this MIADOC directory, both the stuff that is to become publicly avaiable, and the stuff that is internal documentation.

Public

index.html
The main entry point for the public documentation
new.html
Ole's new proposal for the main entry point for the public documentation. To be renamed to index.html if agreed upon.

Private/Internal

Internal Documentation
Internal documentation with TODO lists, CHANGES files, links to internal documentation to all tools, libraries, statistics etc. Not to be exported. Password protected.
This file
Not to be exported.
Admin
Administrative stuff for MIADOC, e.g., Document-List, Rules, ...
Not to be exported.
Templates
Templates for frontpage, navigation buttons, index etc. Not to be exported.

Misc

There are some other directories, which there are no direct links to from the public documents. They are mentioned here for completenes:
bin/
Contains various scripts used to maintain this documentation
javascript/
Contains scripts used in the manuals, e.g. a script to display the last modification time of each file.
images/
Contains a number of images used in more than one manual, e.g. the navigation buttons. Do not put images here, that are specific to a given manual (e.g. screen dumps).
style/
Contains style sheets used in these manuals.
