Compile/Run Menu

This menu is only present in the menu bar, if the language supported in the active code editor is BETA.

Check Current
The current fragment group in the browser is checked. Output from the checking is written in the Info pane. Possible semantic errors are shown in semantic error viewer and the structure corresponding to the firs semantic error in the list is selected.
Recheck
The fragment group that was previously checked or compiled is checked.
Compile Current
The current fragment group in the browser is compiled, i.e. checking, code generation and linking is performed. Output is written in console were Mjolner was started. Possible semantic errors are shown in semantic error viewer and the structure corresponding to the first semantic error in the list is selected.
Recompile
The fragment group that was previously checked or compiled is compiled.
Run Current
If an executable exists that has the same name as the current fragment group it is started.
Rerun
The application that was previously run is started again. You are asked whether you want to kill the old process.
Compile and Run
Combines the to operations.
Recompile and Run
Combines the to operations.
Quit Current
Kills the process that was started by Mjolner.
Semantic Errors...
Opens the semantic error viewer.
Next Semantic Errors
Selects the next semantic error.
Debug
Activates the debugger on the current program in the browser.
Redebug
Restarts the debugger on the previously debugged program.
