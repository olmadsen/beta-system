Known Bugs and Inconveniences

Windows

Redirecting output through redirectFromFile has not yet been implemented on systems running Windows 95/NT. The same limitation exists for reading and writing to a pipe. Using a pipe to connect to external programshas been implemented, though.

UsageTimeStamp has not yet been implemented. Commpool therefore selects a random socket when choosing a connection to break, not the least recently used.
