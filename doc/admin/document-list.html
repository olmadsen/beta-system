Mjølner System Manual document list

Introduction

The following documents constitute the Mjølner System manuals.

To "allocate" a new MIADOC number simply add the document to this list.

Below each manual is "status" field describing who is responsible for the manual, the title and when it was last updated. Besides this the file format used is noted. Should become HTML for most manuals soon.

For further instructions on how to maintain the manuals, please refer to http://www.mjolner.com/Quality/miadoc.html
