HTML mode for emacs

 The build-in SGML mode for emacs can be used for editing HTML as well. I prefer another version instead. You can use it by copying the following
  • html-mode.el
to your home directory and adding something like 
(autoload 'html-mode "html-mode" "HTML major mode." t)
(or (assoc "\\.html$" auto-mode-alist)
  (setq auto-mode-alist (cons '("\\.html$" . html-mode) 
                              auto-mode-alist)))
into your .emacs file. I have also added this: 
(defun myhtml ()
  (interactive)
  (local-set-key [f1]   (function (lambda () (interactive)(html-add-code 1))))
  (local-set-key [f2]   (function (lambda () (interactive)(html-add-bold 1))))
  (local-set-key [f3]   (function (lambda () (interactive)(html-add-emphasis 1))))
)
(setq html-mode-hook 'myhtml)
which allows me to select something in the HTML and then making it code, bold or emphazised by using F1, F2 or F3, respectively.

Emacs Macros for index generation

To help generation of indexes there is two macros in the file make_index_entry.el.

Load the file into emacs and evaluate the bugger using "M-x eval-buffer". You can now assign these macros to some key-combination. Here is how to bind to "F7" and "F8":
"M-x global-set-key", press F7, type "insert-marker". Likewise for insert-href.

An index is in two parts: A name marker in the text-file and a link in the index file. It is assumes that you allready have a index file. (copy template/inx.html). Open the text file in a buffer and the inx file in another. You can see the both using "C-x 2".

Find the next keyword to go into the index. In the index file find the place where the link should be according to the alphabet and make a empty line. Change back to text-buffer. Double-click the word with the mouse and press F7 and the F8. Now there should be a <A NAME=..> in the textfile and a <A HREF=...> in the index file.
