RectShape

A Rect composed of a
RectShape and a Paint:

The rectangle shape is defined by the following three attributes:

UpperLeft.
Point specifying the upper left corner of the rectangle.
Width and Height.
The width and height of the rectangle.

A corresponding Rect graphical object is defined with RectShape as the shape. Rect uses FillRect of the Paint to draw itself (see section 9.8.1).
