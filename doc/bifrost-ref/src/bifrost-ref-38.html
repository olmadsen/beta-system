9 Predefined Shapes and Graphical Objects

9.2 MultiLineShape

A MultiLine composed of a
MultiLineShape and a Paint:

The multi line shape is defined by five attributes:

Points.
PointArray defining the line. A PointArray, like the name indicates, is an array/list of Points. See the interface description for details.
Width and Dashes.
Same as for Line.
Cap.
Specifies how the ends of the line looks. See section 4.5.1.
Join.
Specifies how the lines are joined. See section 4.5.1.

A corresponding MultiLine graphical object is defined with MultiLineShape as the shape. MultiLine uses FillMultiLine of the Paint to draw itself (see section 9.8.1).
