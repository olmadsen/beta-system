12 Bifrost and Lidskjalv

As has been mentioned in the previous chapters, the current implementation of Bifrost is based on the Lidskjalv User Interface Toolkit, also known as guienv, see [MIA 94-27]. As it was also mentioned in a footnote in the Canvas description, the Canvas pattern is in the current implementation named BifrostCanvas. This chapter tries to give an overview of the current situation with respect to such overlaps and inconsistencies between Lidskjalv and Bifrost.

The Lidskjalv library and the Bifrost library has been designed independently. This is the reason that there is some overlap in functionality, in the implementation of Bifrost under Lidskjalv.
