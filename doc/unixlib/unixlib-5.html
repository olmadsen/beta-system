The unixFile Library

Using the unixFile Fragment

A program using the unixFile fragment will have the following structure:


INCLUDE '~beta/unixlib/unixFile'
--- program: descriptor ---
(# ...
uf: @unixFile;
do ...
('a','r') -> uf.entry.permission.add;
...
#)
