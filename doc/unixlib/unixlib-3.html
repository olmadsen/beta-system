The UNIX Libraries

This document contains documentation the UNIXlibraries of the Mjølner System. It is a programmer's guide. The purpose is to provide the necessary information to the users of the libraries. The facilities are documented by means of the BETA interfaces and examples of how to use the facilities.

This document includes the documentation on the following libraries of the Mjølner System:

¥ unixFile.bet contins the UNIX specific file system interface.

¥ unixDirectory.bet contains the interface into the hierarchical UNIX file system directories.
