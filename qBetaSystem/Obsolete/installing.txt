== Check Out and Compile ==

* Check out *

  cd c:
  mkdir beta
  cd beta
  svn checkout http://svn.nfit.au.dk/svn/beta-bin_ntiwrapper/trunk bin
  svn checkout http://svn.nfit.au.dk/svn/beta/trunk r5.5
  cd r5.5
  svn checkout http://svn.nfit.au.dk/svn/beta-postwimp/trunk postwimp


* Boot *

  mbs_boottrack

* Compile

  cd $BETALIB/MiniSystem/miniEnv
  beta -qw minienv
