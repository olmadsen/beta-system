Object-Oriented Language Interoperability

This page will eventually describe the Object-Oriented Language Interoperability with .NET and Java, which is currently being implemented for the BETA programming language.

Progress

[2004-05-12]
We have made a new release!
This new release mainly deals with ROTOR: The compiler now detects if it is running under ROTOR and will generate a script that invokes the clix launcher for the generated .NET executable. Furthermore some errors that prevented some programs from working under ROTOR have been fixed. Finally the documentation has been slightly improved, although a lot is still to be done there.
