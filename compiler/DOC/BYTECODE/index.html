Object-Oriented Language Interoperability

This page will eventually describe the Object-Oriented Language Interoperability, which is currently being implemented for the BETA programming language.

Progress

[2003-08-29]
The documentation below as well as the preliminary systems you may download below are currently rather old.
This does not mean that there is no progress at the moment!
The current activities are focused on bringing the compiler into a state where it can bootstrap itself. This work involves implementing a large number of the BETA libraries using the Java and .NET Framework classes. This is close to being completed for Java, whereas the .NET library port is far from complete. The BETA compiler for Java/JVM - jbeta - can currently be compiled and started, but a number of Java verification errors must be corrected before the bootstrapped compiler will eventually run.
We expect to make a new package when both the JVM and .NET based compilers can boot themselves.
