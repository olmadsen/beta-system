Gennemgå helloapplet.bet   4.  jbeta helloapplet.bet   5.  Gennemgå helloapplet.html   6.  Åbn helloapplet.html i Internet ExplorerB. Nedarvning af BETA i extern klasse   1.  make clean   2.  Gennemgå Bclass.bet   3.  Gennemgå BclassUser.cs   4.  nbeta -x Bclass.bet   5.  csc /nologo /r:System.dll /r:dotnet/tstenv.dll /r:dotnet/Bclass.dllBclassUser.cs   6.  BclassUser.exe   7.  Gennemgå BclassUser.java   8.  jbeta -x Bclass.bet   9.  javac -d javabc -classpath javabc BclassUser.java   10. java -classpath javabc BclassUserC.
