Compiler Internal Documentation

The files in these documents document a various aspects of the BETA compiler. There is no systematic documentation for the whole compiler. The current files have been collected from various sources and they may NOT be up-to-date. Over time the documentation will be extended and brought up-to-date.

The paper: 

[MjolnerBook-CompilerDoc] O.L. Madsen, 
			  The Implementation of BETA,
			  In: Object-Oriented Environments - 
			  The Mjolner Approach,
			  Prentice Hall, 1994
should be read first. There are, however, things that have been changed, since that paper was written.
